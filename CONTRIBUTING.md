# Guía de Contribución

¡Gracias por tu interés en contribuir a **AI Flow**! Este documento explica cómo puedes ayudar a mejorar el proyecto.

## 🛠 Cómo Contribuir

### 1. Reportar Errores
- Abre un [nuevo Issue](https://github.com/tu-usuario/ai-flow/issues) con la etiqueta `bug`
- Usa el formato:
  ```markdown
  ### Descripción
  [Explica el error claramente]

  ### Pasos para reproducir
  1. ...
  2. ...

  ### Comportamiento esperado
  [¿Qué debería pasar?]

  ### Screenshots/logs (opcional)
